# Trust or Trap — 기획안 (에이전트 참조용)

> 몰입형 피싱/스캠 예방 시뮬레이션 게임. 정답형 퀴즈가 아니라 **판단형** 학습.
> 상세 스토리 시놉시스·통계는 원본 기획 문서 기준. 이 파일은 **개발 시 항상 참조할 핵심 결정**만 정리.

## 한 줄 정의

사용자 프로필(나이·성별)에 맞는 범죄 시나리오를 **"폰 속의 폰" UI**로 체험하고, AI 대화 + 분기 선택으로 위험 신호를 학습하는 게임.

## 차별화

- 페르소나 매칭 스토리 (객관식 정답 X)
- Scriptic 스타일 실제 앱 UI 재현 (카톡/문자/인스타/전화/은행)
- 시나리오 그래프(고정) + LLM(표현만 생성)

## 전체 플로우

```
[1] 온보딩: 이름(가명)/나이/성별
[2] 페르소나 매칭 → 기본 시나리오 추천 + [다른 사건 선택]
[3] 폰 속의 폰 진입 → 알림으로 스토리 시작
[4] 인터랙션: 자유 선택/입력 → AI 위험도 판정 → 분기 (타이머 압박)
[5] 엔딩 3갈래: 안전 / 경고(피해 후 대응) / 심화(피해)
[6] 사후 리포트: 놓친 위험 신호 리플레이
```

## 페르소나 × 기본 시나리오 (7종)

| 입력 | 기본 매칭 | 시나리오 ID |
|------|-----------|-------------|
| 10대 여성 | 디지털 성범죄 (그루밍) | `teen-female-grooming` |
| 10대 남성 | 게임 아이템 사기/몸캠피싱 | `teen-male-game-scam` |
| 20대 여성 | 로맨스 스캠 | `twenties-female-romance` |
| 20대 남성 | 알바위장 대출빙자형 보이스피싱 | `twenties-male-voice-phishing` |
| 30~40대 | 투자리딩방 | `midlife-investment-scam` |
| 50대 | 대환대출·정책자금 빙자형 | `fifties-loan-phishing` |
| 60대+ | 기관사칭형 보이스피싱 | `senior-institution-phishing` |

대체 선택지: 투자리딩방 등은 항상 [다른 사건 선택]에 노출.

## 시나리오 공통 노드 패턴

```
접근 → 신뢰형성 → 위험신호제시 → 요구 → 분기(안전/경고/피해)
```

## 엔딩 3갈래

| 타입 | UI 색 | 예 |
|------|-------|-----|
| `safe` | green | 거절 & 신고 |
| `warn` | yellow | 소액 피해 → 대응 안내 |
| `danger` | red | 피해 심화 → 사후 절차 |

## 10대 시나리오 특수 규칙

- **음성(TTS/STT) 미적용** — 텍스트/이미지 채팅만
- 그루밍: 대화 경과일 표시 (~102일 체감)
- 금지: 실제 성적 콘텐츠, 미성년자 음성 데이터 수집

## UI 앱 종류

`ChatApp` | `SMSApp` | `CallScreen` | `BankApp` | `InstagramDM` | `DatingApp`

## 참고 유사 서비스

블루 메신저, 보이스피싱크홀, Scriptic Crime Stories (UI 참고)
