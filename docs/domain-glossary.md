## Risk Flags

| Flag | Meaning |
|------|---------|
| `safe` | 올바른 대응 → safe ending path |
| `warn` | 부분적 실수 → warn ending |
| `danger` | 위험 행동 → danger ending |
| `neutral` | 아직 판정 전 / 중립 진행 |
